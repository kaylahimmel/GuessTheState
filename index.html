Guess the State

A hangman-style game

The answer to each puzzle is a state name, so only letters will be used. To start, guess any letter. If you choose a correct letter, it will appear in the puzzle. If the letter you choose is not used in the puzzle, your remaining guesses will decrease by one. You will win by guessing all of the correct letters before the remaining guesses teaches zero.
